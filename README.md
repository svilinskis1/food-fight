# Food Fight
## by Samera Vilinskis

Uh oh! Due to inflation, your restaurant is going under! Luckily, there are some less ethical ways to get food- the nuclear waste sewers! Fight monsters to save enough cash to save your restaurant! 

A little personal project to try out [PICO-8](https://www.lexaloffle.com/pico-8.php). Originally made February - June 2023. All code, art, sprites, and sound effects created by me.
